Introduction

HTML is an acronmy for hyper text markup language,and it is the standard language for creating web-page. every web-page you surf on your browser is built using HTML to provide a proper structure to the web-page so all the content or data of the page could represent adaquately. A stand alone HTML can create only static and skelaton looking black and white pages,but with the help of CSS and JavaScript ,We can create a more interavtive and intuitive web-page.When wwe try to visit a website or click on the link, We basically request the server to show us the page, Then the server acts on our request and send us an appropriate HTML. HTML Elements

The HTML Elements provides the semantic meaning to the web-page content. We usually interchangeably use the term HTML elements and tags, but technically both are different. An HTML tag is just a chracter inside the angle bracket <>, Where as the HTML elements is a collection of starting tag, its attribute , content and end tag. For example...

<class="para">Hello World </p>

HTML Heading

To dispaly the section heading, title or subtitle We can use the HTML heading tags. In HTML 5 We have 6 heading tags start from <h1> upto <h6>,Where <h1> specify the largest heading and <h6> represents the smallest or the sub heading. if the content is specified by heading tags,Then it would be displayed large and bold as compared to other text content present on the web-page.Example 

 
                            <h1>First Heading </h1>
                            <h2>Second Heading </h2>
                            <h3>Third Heading </h3>
                            <h4>Forth Heading </h4>
                            <h5>Fifth Heading </h5>
                            <h6>Sixth Heading </h6>

HTML Formatting

  • In,HTML we have many special elements that can provide special meaning to text content.
HTML ELEMENTS DESCRIPTION
<b></b> Bold the text
<strong></strong> an alternative for <b>, Which tells that text is essential.
<i></i> italic the text.
<em></em> Similar to italic but used when we want to emphasize the text.
<mark></mark> It mark the text with a default "Yellow" background color.
<small></small> Decrease the text size.
<del></del> it print a cross line over the text.
<ins></ins> Represents the inserted text by putting an underline.
<sub></sub> this element is used to display the subscript.
<sup></sup> it can make a text superscript.
